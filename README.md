# JJK RNG (fan game)

Браузерная RNG-игра **по мотивам Jujutsu Kaisen (JJK)**: крутки, редкости, pity, коллекция, улучшения и задания.

## Запуск

- Открой `index.html` двойным кликом.
- Прогресс сохраняется в `localStorage` браузера.

## Управление

- **Крутка**: трать “Кубы” и получай персонажей/ресурсы.
- **Коллекция**: смотри кого уже выбил, их редкость и дубликаты.
- **Улучшения**: усиливай шанс редких дропов и заработок кубов.
- **Задания**: ежедневные цели на кубы.

## Дисклеймер

Это фан-проект. Все названия/образы относятся к их правообладателям. Проект некоммерческий.
